# **Overview**

UDP (Unity Distribution Portal) is a Unity product that allows you to distribute your games to multiple app stores through a single hub. UDP repacks your Android build with each storeâ€™s dedicated IAP SDK and lets you manage all your store submissions from the UDP console.

### Prerequisites
* UDP is supported in Unity 5.6.1 or higher. 
* UDP currently only supports Android.
